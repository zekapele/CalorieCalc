# calorie counter

Трекер харчування на C++/Qt з простим GUI: пошук продуктів, 4 секції прийомів, цілі та статистика, профілі, календар і збереження по днях.

## Можливості
- Пошук продуктів (вбудована база + ваші кастомні продукти)
- Сніданок / Обід / Вечеря / Перекус (окремі вкладки)
- Ціль калорій, підсумки за день, прогресбар калорій
- Вода: ціль і споживання (мл), прогресбар
- Вага: фіксація ваги на обраний день
- Шаблони прийомів їжі (зберегти/додати для кожної вкладки)
- Профілі (у т.ч. «Без профілю») і календар днів


## Встановлення
Вимоги: CMake 3.10+, компілятор C++20, Qt5/Qt6 Widgets.

```bash
mkdir build && cd build
cmake -DBUILD_GUI=ON ..
make -j
```

macOS (app bundle):
- збирання створює `calorie counter.app` (у `build/` або `cmake-build-*`)
- для деплойменту: `macdeployqt "calorie counter.app"`

## Запуск
```bash
# Консольна версія (CLI)
./CalorieCalc

# Графічна версія
./CalorieCalcGUI            
```

## Як користуватись
- Ліворуч: пошук, кількість (г), тип прийому, кнопка «Додати», блок «Додати свій продукт»
- Праворуч: вкладки прийомів, кнопки шаблонів, видалення, цілі та статистика, прогресбари
- Зверху: профіль (редагований) та календар — перемикають сховище на інший день/профіль

## Стек і архітектура
- C++20, Qt Widgets
- ООП: `Food`, `SavedMeal`, `Diary`, `FoodDatabase`
- Strategy: `ISaveStrategy` + `JsonSaveStrategy`/`TextSaveStrategy`
- Збереження: JSON файли по датах і профілях
